Como usar o Ranking de Economia

💰 O que é a Cota CEAP?
A Cota para o Exercício da Atividade Parlamentar (CEAP) é uma verba mensal de até R$45.000 que cada deputado recebe para custear atividades como passagens, hospedagem, telefone e refeicões. O dinheiro é público.
🏆 Pódio Top 3
Os três deputados que menos gastaram da cota no período selecionado aparecem no pódio com medalhas de ouro, prata e bronze. Um baixo gasto significa mais economia com o dinheiro público.
🔎 Filtros
Use os filtros para ver o ranking de um Estado específico, de um Partido ou de um ano diferente. O ranking recalcula automaticamente após cada filtro.
⏳ Carregamento Progressivo
O sistema busca os gastos de todos os 513 deputados em lotes, o que pode levar alguns minutos na primeira vez. Uma barra de progresso indica quantos já foram processados. As visitas seguintes carregam instantâneamente pelo cache.
🔗 Validação Oficial
Os dados vêm diretamente da API da Câmara dos Deputados. O link "Portal da Transpariência Oficial" no topo da tabela leva ao site oficial para validação independente.
