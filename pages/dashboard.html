Como usar o Dashboard

📊 KPIs no topo
Os 4 cartões mostram o total de deputados ativos, o gasto acumulado da cota CEAP no ano, o número de votações realizadas e a quantidade de partidos com bancada. Os valores são carregados da API oficial da Câmara.
🍥 Gráfico de Distribuição Partidária
Mostra todos os partidos com assento na legislatura atual. Cada fatia usa a cor oficial do partido. Passe o mouse sobre uma fatia para ver o nome completo e o número de deputados.
⏳ Carregamento de dados
Alguns KPIs dependem de cálculos pesados (ex: total CEAP). Se demorar a carregar, aguarde — os dados estão sendo buscados diretamente da API oficial e colocados em cache para visitas futuras serem instantâneas.
🔗 Navegação
Use a barra lateral esquerda para acessar as outras seções. Clique no logo no topo para voltar à página inicial.
